Customer Shopping Behaviour Analysis
Tools & Technologies: Python, SQL, Power BI, Excel
•	Analysed 3,900+ customer transactions across 50 locations to identify spending patterns, customer segments, and key revenue drivers.
•	Performed data cleaning and preprocessing, handling missing values and standardizing features to ensure accurate exploratory data analysis (EDA).
•	Conducted customer segmentation analysis, identifying 80% loyal customers, 18% returning, and 2% new users, highlighting retention and growth opportunities.
•	Evaluated subscription performance, finding similar average spend between subscribers and non-subscribers, indicating scope for improving subscription value propositions.
•	Delivered actionable recommendations on loyalty programs, discount optimization, and targeted marketing strategies based on demographic and behavioural insights.
